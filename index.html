UNEG | Validación de formularios

Esta aplicación web consiste en una página que contiene dos formularios distintos para demostrar diferentes métodos de validación. El primer formulario utiliza validaciones básicas de HTML y JavaScript para asegurar que los datos ingresados sean correctos antes de ser enviados. El segundo formulario emplea Bootstrap para validar los campos automáticamente siguiendo las reglas predefinidas. Además, se incluye una breve descripción sobre los métodos de validación utilizados y una lista de pasos para crear y desplegar la aplicación.

¿Cómo hice esta aplicación? Paso a paso

  • Estructura básica del documento HTML
    • Creación de la estructura básica del documento HTML con etiquetas <html>, <head>, y <body>.
  • Añadir metadatos y enlaces de CSS
    • Incluir metadatos como charset, viewport, descripción y autor.
    • Establecer el título de la página.
    • Enlazar a la hoja de estilos de Bootstrap y a un archivo CSS personalizado.
  • Estructurar el cuerpo de la página
    • Crear un contenedor principal que ocupa toda la altura de la ventana.
    • Añadir una barra de navegación con un logo y el título de la página.
    • Incluir un contenedor para el contenido principal con dos formularios y una tarjeta explicativa.
    • Añadir un pie de página con información del autor.
  • Primer formulario (HTML y JS)
    • Crear un formulario con campos validados usando HTML.
    • Añadir campos adicionales validados con JavaScript.
    • Incluir un botón de envío para validar el formulario.
  • Segundo formulario (Bootstrap)
    • Crear un formulario con validaciones de Bootstrap.
    • Incluir campos para dirección, país, estado y código postal.
    • Añadir un botón de envío para validar el formulario.
  • Tarjeta explicativa
    • Incluir una tarjeta que explica los diferentes métodos de validación utilizados.
    • Añadir una lista de pasos para la creación y despliegue de la aplicación.
  • Pie de página
    • Crear una barra fija al fondo con información del autor.
  • Aplicar estilos personalizados (CSS)
    • Definir estilos para elementos de entrada, mensajes de error y el pie de página.
  • Validación del segundo formulario
    • Añadir un bloque de código JavaScript para habilitar la validación de Bootstrap.
    • Evitar el envío del formulario si no cumple con las reglas de validación.
  • Validación del primer formulario
    • Añadir un bloque de código JavaScript que se ejecuta cuando el contenido del DOM está completamente cargado.
    • Definir funciones para validar el formato del correo electrónico y el nombre de usuario.
    • Mostrar y ocultar mensajes de error según sea necesario.
    • Validar los campos en tiempo real y al enviar el formulario.
  • Llenado de opciones de estados de Venezuela
    • Añadir dinámicamente las opciones del campo "Estado" en el segundo formulario usando un arreglo de estados predefinido.
